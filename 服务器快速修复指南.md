# 服务器快速修复指南

## 问题：git pull gitee main 失败

### 解决方案1：添加 Gitee 远程仓库

```bash
cd ~/microtrace

# 添加 gitee 远程仓库
git remote add gitee https://gitee.com/bupoo/MicroTrace.git

# 拉取最新代码
git pull gitee main
```

### 解决方案2：直接从 Gitee 拉取（如果方案1失败）

```bash
cd ~/microtrace

# 直接拉取（不指定远程）
git pull https://gitee.com/bupoo/MicroTrace.git main
```

### 解决方案3：重新克隆（最简单）

如果上面的方法都不行，可以重新克隆：

```bash
cd ~
rm -rf microtrace
git clone https://gitee.com/bupoo/MicroTrace.git microtrace
cd microtrace
```

然后继续执行：
```bash
npm install
npm run build
npm install -g pm2 --registry=https://registry.npmmirror.com
pm2 start npm --name "microtrace" -- start
pm2 save
```
