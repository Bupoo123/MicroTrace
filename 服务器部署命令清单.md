# 服务器部署命令清单

## 已完成的步骤
✅ 已连接到服务器

## 接下来执行的命令

### 1. 从 Gitee 克隆代码

```bash
cd ~
git clone https://gitee.com/bupoo/MicroTrace.git microtrace
cd microtrace
```

### 2. 运行部署脚本

```bash
chmod +x deploy-server.sh
./deploy-server.sh
```

### 3. 等待部署完成

脚本会自动完成所有步骤，大约需要 5-10 分钟。

### 4. 部署完成后访问

打开浏览器访问：**http://121.36.255.207:5005**

使用账号：`admin` / `123456` 登录

## 如果遇到问题

### Node.js 安装失败

如果脚本在安装 Node.js 时失败，可以手动安装：

```bash
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 配置npm镜像
npm config set registry https://registry.npmmirror.com

# 然后继续运行部署脚本
./deploy-server.sh
```

### 查看部署日志

如果部署过程中出现问题，可以查看详细输出，或者：

```bash
# 查看 PM2 日志（部署完成后）
pm2 logs microtrace

# 查看应用状态
pm2 status
```
